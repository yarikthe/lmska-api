  • Response

Query Parameters

  • email: required(string)

    Email address

    Example:

    V@rte.im
  • password: required(string)

    Password

    Example:

    123456
  • telephone_number: required(string)

    Phone number

    Example:

    980009999
  • telephone_code: required(string)

    Phone number

    Example:

    38
  • first_name: required(string)

    First Name

    Example:

    John
  • last_name: required(string)

    Last Name

    Example:

    Doe
  • role_id: required(int)

    Role ID

    Example:

    5 (Student)

Body

Media type: application/json

Type: object

Example:

result:

{
  "success": true,
  "message": "User was successfully registered but not verified"
}
